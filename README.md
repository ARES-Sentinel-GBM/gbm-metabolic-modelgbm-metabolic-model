# 🧠 Linkq1-crypto GBM Metabolic Model
## Exploring how the microenvironment shapes Glioblastoma response

This repository contains a computational study on **IDH1-mutant Glioblastoma** metabolic eradication using the Recon3D framework.

### 🔬 Project Overview (Simplified)
Glioblastoma is highly resilient because it can shift its metabolism to survive nutrient deprivation. Our research identifies a multi-target inhibition strategy named **Protocol OMEGA**.

### 🚀 The OMEGA Protocol
By targeting 8 key metabolic checkpoints simultaneously, we achieved a **99.9% reduction in biomass flux**, effectively bringing tumor growth to a functional halt.

#### Targets included:
1. **Glycolysis** (HEX1, GAPD, LDH_L)
2. **Mitochondrial Respiration** (ATPS3m, CYOORm)
3. **Glutaminolysis** (GLS)
4. **Fatty Acid Oxidation** (CPT1, FACOAL)

### 📊 Results
We successfully differentiated between actual metabolic flux and software-related maintenance (NGAM). Under total exchange deprivation, the model reached **Zero Flux**, confirming the theoretical lethality of the approach.
